# Release Notes — v0.5 Analytics Dashboard

## Overview

We are excited to announce the release of the **Analytics Dashboard** for GitPulse! This milestone (v0.5) bridges the gap between raw standup summaries and long-term developer insights, allowing you to visualize your "Git Pulse" directly from the web interface.

## New Features

### 📊 Interactive Commit Frequency

A brand new line-and-bar chart visualization that maps your commit volume over time. 

- Filter by **7, 14, 30, or 90 day** lookback windows.
- Hover over data points for precise commit counts.

### 🥧 Repository Activity Breakdown

Understand where your focus is going across multiple projects.

- Dynamic pie charts visualizing commit distribution by repository.
- Automatic percentage calculation based on actual GitHub activity.

### 🔥 Productivity Insights Panel

A high-level metric dashboard providing immediate "at-a-glance" stats:

- **Longest Streak**: Track your consecutive days of coding activity.
- **Top Repository**: Identify your most active project in the current window.
- **Most Active Day**: Discover which day of the week you are most productive.
- **Avg. Commits/Day**: Data-driven velocity tracking.

## Technical Improvements

- **Direct GitHub Aggregation**: Analytics are generated on-the-fly by querying the GitHub API for precise, real-world data.
- **PostgreSQL Integration**: Fuses local summary history from your Neon DB with live git metrics.
- **Recharts Integration**: High-performance SVG-based charts with built-in responsive scaling.
- **FastAPI Analytics API**: Three new backend endpoints dedicated to serving structured data for visualizations.

## How to Access

Once your local servers are running, navigate to the new route:
`http://localhost:3000/analytics?username=YOUR_GITHUB_USERNAME`
